# Flight Booking App

A minimal **Ionic + Vue** mobile app boilerplate for UI/UX handover and future backend API integration.

## Live Demo

🌐 https://flightbookingapp-seven.vercel.app

## Features

- Mobile-first project structure
- Single root page (`/`)
- SCSS/SASS-ready architecture
- Reusable components and utilities

## Getting Started

```bash
npm install
npm run dev
```

## Quality Checks

```bash
npm run type-check
npm run lint
npm run test:unit
npm run build
```

## Project Structure

- `src/pages` – App pages
- `src/components` – Reusable UI components
- `src/styles` – Global styles & design tokens
